## Snippet-generator mdx文档模板

```js
"Init mdx file": {
"prefix": "imdx",
"body": [
"---",
"title: $1",
    "date: $CURRENT_YEAR-$CURRENT_MONTH-$CURRENT_DATE",
    "lastmod: $CURRENT_YEAR-$CURRENT_MONTH-$CURRENT_DATE",
"tags: ['$2']",
"draft: false",
"summary: $3",
"layout: PostSimple",
"---",
"",
"",
"$0"
],
"description": "Init mdx file"
}
```
