## nginx conf设置

```shell
        location /tensorboard {
            proxy_http_version 1.1;
            proxy_pass http://127.0.0.1:6006/tensorboard;
        }
```

## tensorboard 启动

```sh
tensorboard --path_prefix=/tensorboard --logdir model
```

## 访问

`IP/tensorboard/`,注意需要后面的`/`符号.
